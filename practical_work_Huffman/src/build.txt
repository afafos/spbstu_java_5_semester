Сборка приложения

Требования 
- Должна быть установлена Java Development Kit (JDK)

Этапы сборки
- Откройте командную строку или терминал
- Перейдите в каталог, в который вы хотите склонировать репозиторий
- Клонируйте репозиторий с помощью команды Git:
  git clone 
- Выполните следующие команды для компиляции кода:
  javac HuffmanNode.java
  javac HuffmanCoding.java
