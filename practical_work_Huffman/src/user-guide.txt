Запуск приложения

- Поместите файл input.txt, который вы хотите закодировать, в папку input_output_files внутри директории проекта.

- Откройте командную строку (или терминал) внутри директории проекта.

- После успешной компиляции выполните запуск приложения:
  java HuffmanCoding

После выполнения приложения убедитесь, что в папке input_output_files созданы два файла: 
encoded.bin и decoded.txt. Файл encoded.bin содержит закодированные данные, 
а decoded.txt - раскодированные данные.

Также после выполнения приложения вы увидите результаты в консоли, 
такие как размер оригинального файла, размер закодированного файла, 
размер декодированного файла и коэффициент сжатия.
